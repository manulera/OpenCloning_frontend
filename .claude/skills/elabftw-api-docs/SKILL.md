# eLabFTW API Documentation

## Documentation Location

When using eLabFTW as the database, API endpoints are documented at:

**https://doc.elabftw.net/api/elabapi-html/**

This is the official eLabFTW API documentation. Use it when:
- Looking up eLabFTW-specific endpoints (experiments, items, etc.)
- Implementing or debugging database submission flows with eLabFTW
- Understanding eLabFTW authentication and request formats

This URL is hosted online and does not require a local eLabFTW instance to view the documentation.
